Celery基础

介绍

Celery 是一个简单、灵活、可靠的分布式系统，可处理大量消息，同时为操作提供维护此类系统所需的工具。它是一个专注于实时处理的任务队列，同时也支持任务调度。

📖 官方文档：[Celery 5.2.0](https://docs.celeryproject.org/en/v5.2.0/#) 如果你刚开始学习，参考文档[新手教程](https://docs.celeryproject.org/en/v5.2.0/getting-started/first-steps-with-celery.html)

  • 安装Celery

    pip install Celery

  • 启动Celery工作服务进程

    celery -A xxxx worker --loglevel=INFO

    (从4.0开始不再支持windows，如果启动后报错The operation timed out.，则需要安装pip install eventlet 在启动命令中添加 -P enentlet 或者 - - pool=solo)

  • 配置

    1. 可以在应用实例上直接设置

      app = Celery(``backend=xxx, broker=xxx, include=xxx)

    2. 可以直接设置值

    3. 使用配置文件,

      app.config_from_object('celeryconfig')

      config_from_object() 方法接受模块名称，也接受实际传入的模块对象。具体参考官方文档

      ⚠️ 需要注意的是，新版的参数由原来的大写改成了小写，如果你参考网上教程有些是大写的，那需要所有的都大写，不能大小写混用。
 </aside>

 以下参考[配置含义和默认值](https://docs.celeryproject.org/en/v5.2.3/userguide/configuration.html#configuration)
