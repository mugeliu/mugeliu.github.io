Celery动态添加定时任务

🖇️ Celery添加定时任务后，如果想要更改，就必须更改完重启定时任务。如果想不重启直接添加更改定时任务。就需要自定义调度程度。
  • RedBeat 基于reids的调度程序
  • celerybeat-mongo 基于mongo的自定义调度

使用RedBeat

  1. github
  2. 文档地址
  3. 相关文章

  • 安装

    pip install celery-redbeat

  • 在celery中设置调度地址

    redbeat_redis_url = "redis://localhost:6379/1"

  • 启动celery beat (如果在配置文件中没有设置beat_scheduler的值，则需要启动时-S参数指定调度程序)

    celery -A proj beat -S redbeat.RedBeatScheduler

  • 启动工作进程

    celery -A proj worker --pool=solo --loglevel=info
